# 政企通 — 科创项目申报智能助手

## 产品介绍手册

---

## 一、产品概述

**政企通** 是一款专为中小科技企业打造的科创项目申报服务平台。系统通过智能聚合各级政府扶持政策、AI 解析政策文本、基于企业档案的自动化匹配、全流程申报管理等核心能力，帮助企业高效发现、评估和申报各类科技扶持项目，不错过每一个政策扶持机会。

### 产品定位

- **目标用户**：中小科技企业、科技园区运营方、政策申报服务机构
- **核心价值**：让政策匹配变得简单高效
- **产品 slogan**：科创项目申报智能助手

---

## 二、产品架构

政企通采用前后端分离的全栈架构：

| 层级 | 技术栈 |
|------|--------|
| 前端框架 | React 18 + TypeScript |
| 构建工具 | Vite 6 |
| 样式方案 | Tailwind CSS |
| 状态管理 | Zustand |
| 路由管理 | React Router v7 |
| 后端框架 | Express.js + TypeScript |
| 数据库 | SQLite（better-sqlite3）|
| 认证方式 | JWT（JSON Web Token）|
| 图标库 | Lucide React |

## 三、核心功能模块

### 模块一：政策雷达（政策库）

政策雷达是系统的政策数据中心，汇聚各级政府发布的科技扶持政策。

**功能特性：**

- **多维度筛选**：按区域（太仓市、苏州市、江苏省、国家级）、归口部门（科技部、工信部、发改委等）、支持方式（资金补贴、税收优惠、人才政策、资质认定、项目扶持）筛选政策
- **关键词搜索**：支持按政策标题快速搜索
- **政策订阅**：用户可以创建多个订阅规则（按区域/部门/类型/关键词），新政策发布时自动提醒
- **新政策提醒**：标记上次访问后的新发布政策，支持一键标记已读
- **分页浏览**：支持分页展示，实时查看政策列表

**覆盖政策类型示例：**
- 科技型中小企业入库
- 高新技术企业认定
- 专精特新中小企业认定
- 创业资助项目
- 领军人才计划
- 高企培育入库

---

### 模块二：AI 政策解析

利用 NLP 技术智能解析政策原文，自动提取结构化信息。

**功能特性：**

- **原文输入**：粘贴政策原始文本，系统自动解析
- **智能提取字段**：
  - 政策标题
  - 归口部门
  - 适用区域
  - 截止时间
  - 补贴金额
  - 支持方式
  - 政策描述
  - 门槛要求（结构化键值对）
  - 申报流程（步骤化展示）
  - 所需材料清单
  - 政策标签
- **一键保存**：解析结果可直接保存为正式政策条目
- **历史记录**：查看所有历史解析记录及状态

---

### 模块三：智能匹配

基于企业档案数据，自动计算政策匹配度，精准推荐最适合的扶持项目。

**功能特性：**

- **企业健康度评分**：综合所有政策匹配结果，计算企业科创健康度（0-100分）
- **匹配度计算**：逐项比对政策要求与企业档案，得出每项政策的匹配百分比
- **匹配详情展示**：
  - 已匹配项（绿色标签）
  - 待提升项（红色标签，含改进建议）
  - 补贴金额展示
  - 截止日期提醒
- **排序筛选**：按匹配率升降序排列，支持搜索过滤
- **一键跳转**：从匹配结果可直接进入政策详情或申报工作台

**匹配逻辑依据的企业数据维度：**
- 企业营收规模
- 员工数量及社保缴纳情况
- 知识产权拥有量（专利、软著等）
- 所属行业领域
- 注册区域
- 团队学历构成

---

### 模块四：企业档案

结构化存储企业全量信息，一次录入、永久复用。

**功能包含五个子模块：**

#### 1. 基本信息
- 企业全称、统一社会信用代码、法人代表
- 企业类型（有限责任公司、股份有限公司等）
- 注册资本、成立日期
- 所属行业（软件/信息技术、人工智能、生物医药等 9 大行业）
- 所属区域、员工总数、社保缴纳人数
- 营业收入、经营范围、企业简介

#### 2. 资质证照
- 管理各类资质证书（营业执照、专利证书、高企证书、ISO认证等）
- 支持记录：证书编号、发证机关、颁发日期、有效期
- 过期自动标识
- 支持上传附件（PDF/Word/Excel/图片，最大20MB）

#### 3. 财务资料
- 管理审计报告、纳税证明、财务报表等
- 支持记录出具单位、出具日期、年份期间
- 支持附件上传

#### 4. 合同协议
- 管理销售合同、技术开发合同、产学研协议等
- 支持记录合同金额、合作方、签订日期、合同类型
- 支持附件上传

#### 5. 团队成员
- 管理核心团队成员信息
- 支持记录：姓名、职位、学历、职称、专业背景、入职日期、联系方式
- 社保缴纳状态标识
- 数据看板展示各模块数量统计

---

### 模块五：申报工作台

全流程任务管理，跟踪申报进度。

**功能特性：**

- **任务分类**：按材料、资质、财务等分类管理
- **状态流转**：待办 → 进行中 → 已完成，支持任务状态变更
- **进度追踪**：
  - 进行中任务显示进度条
  - 截止日期倒计时提醒
- **筛选视图**：全部 / 待办 / 进行中 / 已完成（含数量统计）
- **自定义任务**：支持手动添加、编辑和删除任务
- **已完成任务**：显示完成状态，支持重置

---

### 模块六：爬虫管理

自动化政策抓取系统，从政府网站自动获取最新政策。

**功能特性：**

- **政策源管理**：
  - 添加政府部门网站作为政策来源
  - 配置名称、网址、归口部门、区域
  - 启用/禁用政策源
- **手动触发爬取**：支持单源爬取和全量爬取
- **爬取日志**：记录每次爬取的政策标题、状态、错误信息
- **自动入库**：爬取成功的政策自动进入政策库

**预置政策源示例：**
- 太仓市人民政府
- 苏州市科技局
- 江苏省工信厅

---

### 模块七：资源库

集中管理政府网站和申报平台链接。

**功能特性：**

- **链接管理**：添加、编辑、删除常用政府网站链接
- **分类整理**：按自定义分类组织（政府门户、申报平台、通知公告等）
- **快速访问**：一键跳转目标网站
- **搜索过滤**：按名称/链接搜索，按分类筛选
- **信息备注**：为每个链接添加备注说明

---

### 模块八：仪表盘

一站式工作台，概览企业科创申报全局。

**功能特性：**

- **科创健康度**：环形图展示企业综合健康评分（优秀/良好/待提升）
- **短板诊断**：自动识别未满足的政策条件，给出改进建议
- **推荐项目**：按匹配度排列，展示 Top 推荐政策
- **待办事项**：展示当前待处理申报任务
- **申报日历**：日历组件展示申报截止日期
- **快捷入口**：一键跳转政策库、企业档案、智能匹配、申报工作台
- **新政策通知**：订阅政策更新时自动推送通知

---

### 模块九：园区驾驶舱（管理员）

面向园区运营方的数据管理与决策支持平台。

**功能特性：**

- **数据总览**：入驻企业数、平均健康度、累计申报数、获得补贴总额
- **企业健康度排名**：按行业筛选，展示各企业健康评分及排名趋势
- **健康度分布**：饼图展示企业健康分布（健康/正常/预警）
- **企业列表**：查看入驻企业详情，支持一键推荐政策
- **免申即享管理**：配置自动享受政策（免申即享），设置适用条件
- **申报动态**：实时展示企业申报行为动态

---

### 模块十：用户系统

**功能特性：**

- **注册/登录**：邮箱注册，JWT 鉴权
- **角色权限**：普通用户 + 管理员双角色
- **自动登录**：Token 持久化，刷新保持登录状态
- **安全退出**：一键退出，清除登录凭证

## 四、使用流程

```
第一步：注册账号 → 第二步：完善档案 → 第三步：智能匹配 → 第四步：申报管理
```

### 详细指引：

**1. 注册与登录**
   - 访问系统首页，点击"免费注册"
   - 填写邮箱、密码、企业名称完成注册
   - 注册成功后自动登录，进入仪表盘

**2. 完善企业档案**
   - 进入「企业档案」页面
   - 填写基本信息（企业全称、行业、区域、营收等）
   - 添加资质证照（专利、软著、认证证书等）
   - 录入团队成员信息（学历、社保状态等）
   - 上传相关证明材料

**3. 浏览与订阅政策**
   - 进入「政策雷达」浏览全部政策
   - 使用筛选器按区域、部门、类型过滤
   - 创建订阅规则，新政策自动推送
   - 查看政策详情，了解申报条件、材料和流程

**4. 智能匹配**
   - 进入「智能匹配」页面
   - 系统自动基于企业档案计算匹配度
   - 查看匹配结果，了解匹配项和待提升项
   - 针对待提升项进行改进（如补充知识产权、完善社保等）

**5. 申报管理**
   - 进入「申报工作台」管理申报任务
   - 查看待办事项，按优先级处理
   - 追踪任务进度，确保按期完成
   - 完成任务后标记完成状态

**6. AI 解析（高级功能）**
   - 进入「AI政策解析」页面
   - 粘贴政策原文
   - 系统自动提取结构化信息
   - 一键保存为正式政策条目

## 五、数据安全

- **加密存储**：用户密码使用 bcrypt 哈希加密
- **JWT 鉴权**：所有 API 请求需携带有效 Token
- **权限控制**：普通用户和管理员分离，管理员功能受限访问
- **数据库安全**：使用参数化查询，防止 SQL 注入
- **文件安全**：上传文件类型和大小双重校验
