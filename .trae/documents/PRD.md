## 1. 产品概述

帮助中小科技企业（特别是软件/AI/安全类）一站式解决政府扶持项目申报难题，实现"政策不错过、资质早规划、申报提效率"。

- 主要目标是将分散在各政府部门的扶持政策进行结构化采集和智能匹配，解决企业信息碎片化、申报窗口期短、资质与政策难以快速匹配的痛点
- 面向10-100人规模的ToB科技企业，以及众创空间/孵化器运营人员，提供从政策发现到申报材料准备的全流程数字化工具

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 企业用户 | 邮箱/手机注册 | 填写企业画像、查看匹配政策、管理申报材料、生成申报清单 |
| 园区管理员 | 管理员分配 | 查看园区企业申报健康度、推荐申报项目、管理企业资质 |

### 2.2 功能模块

1. **首页仪表盘**：企业科创健康度评分、今日推荐项目、申报倒计时日历
2. **政策雷达**：政策多源采集、结构化解析、标签体系、订阅推送
3. **企业数字档案**：工商信息、资质资产库、人员结构管理
4. **智能匹配**：政策门槛与企业画像自动比对、匹配度计算、短板诊断
5. **申报工作台**：任务清单、材料模板库、进度追踪
6. **园区管理后台**（特色功能）：园区驾驶舱、一键推荐、免申即享专区

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页仪表盘 | 健康度评分 | 基于企业资质完整度、政策匹配数等维度综合计算科创健康度分数 |
| 首页仪表盘 | 推荐项目 | 根据企业画像自动推荐高匹配度可申报项目，带红点提示 |
| 首页仪表盘 | 申报日历 | 可视化日历展示各项目申报截止时间节点 |
| 政策库 | 筛选器 | 按归口部门（科技局/工信局）、支持方式（资金/资质）、区域（太仓/苏州/江苏）筛选 |
| 政策库 | 政策列表 | 展示项目名称、截止时间、补贴金额、匹配度进度条 |
| 政策详情 | 政策解读 | 结构化展示政策门槛要求、申报流程、材料清单 |
| 企业档案 | 基础信息 | 工商信息展示（营收、社保人数、成立时间） |
| 企业档案 | 资质资产 | 软著/专利证书管理、高企证书、ISO/CMMI等资质录入 |
| 企业档案 | 人员结构 | 研发人员名单管理、学历、社保状态 |
| 智能匹配 | 匹配结果 | 展示各政策与企业的匹配度，标注达标项和短板项 |
| 申报工作台 | 任务清单 | 针对具体项目生成待办材料清单及截止日期 |
| 申报工作台 | 材料模板 | 内置通用申报材料模板，支持在线编辑和下载 |
| 园区管理 | 园区驾驶舱 | 园区内企业申报健康度总览、即将到期资质提醒 |
| 园区管理 | 一键推荐 | 管理员选中企业推送推荐申报项目 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝色（#1e3a5f）为主色，代表专业与信任；翠绿色（#10b981）作为强调色，代表成长与政策红利
- **按钮风格**：圆角矩形（border-radius: 8px），带轻微阴影和悬浮动效
- **字体**：系统字体优先（-apple-system, 'Noto Sans SC'），兼顾中西文可读性
- **布局风格**：卡片式布局，左侧导航栏 + 右侧内容区，顶部固定导航
- **图标风格**：简洁线性图标（lucide-react），统一2px线宽

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页仪表盘 | 健康度评分 | 圆形进度环、分数数字、等级标签（优秀/良好/待提升） |
| 首页仪表盘 | 推荐项目 | 水平滚动卡片、红点角标、匹配度标签 |
| 首页仪表盘 | 申报日历 | 月视图日历组件、颜色标记不同类型项目 |
| 政策库 | 筛选器 | 下拉选择器、标签多选、搜索输入框 |
| 政策库 | 政策列表 | 卡片列表、匹配进度条（蓝绿渐变）、截止倒计时标签 |
| 企业档案 | 资质资产 | 网格卡片展示证书、上传按钮、OCR识别入口 |
| 智能匹配 | 匹配结果 | 雷达图展示各维度匹配情况、达标/不达标标签列表 |
| 申报工作台 | 任务清单 | 勾选框列表、进度条、分类标签（材料/资质/财务） |

### 4.3 响应式设计

- 桌面端优先设计，最低支持 1366px 宽度
- 平板端（768px-1366px）采用两栏布局调整
- 移动端（<768px）采用单栏堆叠布局，导航折叠为汉堡菜单

### 4.4 3D 场景（不适用）

不涉及 3D 场景。

## 5. 非功能性需求

| 类别 | 要求 |
|------|------|
| 数据更新 | 政策源每日自动更新（定时任务） |
| 安全性 | 企业数据加密存储，敏感信息脱敏展示 |
| 响应速度 | 政策检索响应时间 < 2秒 |
| 部署方式 | 支持云端部署及私有化部署 |
